# Desk Setup Switcher v0.1.0 public beta

Desk Setup Switcher `v0.1.0` is the first public-beta version of a local-only macOS menu bar app for saving and deliberately applying desk profiles.

## Capture → Edit → Review & Apply

- **Capture** readable display, audio, and network settings without changing the Mac.
- **Edit** only the values that belong in the profile.
- **Review & Apply** every planned operation and omission before an explicit Apply.
- Keep profiles and redacted diagnostics on the Mac. There is no account, cloud sync, telemetry, analytics, automatic switching, or in-app updater.

## Support and verification boundary

- The `v0.1.0` support boundary is **Apple Silicon with a macOS 14 Sonoma deployment target**. This claim applies only to the exact signed candidate after that candidate passes the Sonoma lifecycle gate; an unpublished draft does not establish support. Intel is not supported in this beta.
- Display, Audio, and Network apply/rollback paths are deterministic mock verified, not hardware-mutation verified. No live setting mutation is claimed for this beta.
- Rollback is a best-effort attempt and can fail. Inspect every itemized result and verify the current macOS state directly.
- Full assistive-technology certification is not claimed. Keyboard behavior, accessibility names/values, and non-color state cues remain part of the supported UI boundary.

## Install safely

A draft, prerelease candidate, Actions artifact, or ad-hoc DMG is not a supported download. The only supported artifact is the immutable Developer ID-signed, notarized, and stapled DMG attached to a maintainer-approved GitHub Release after the complete [distribution gate](https://github.com/GGULBAE/desk-setup-switcher/blob/v0.1.0/docs/DISTRIBUTION.md) passes.

1. Download the DMG and checksum from the same approved GitHub Release.
2. Verify the published SHA-256 before opening the DMG.
3. Install from the DMG, launch the menu-bar app, and use Capture → Edit → Review & Apply. Capture reads without applying; only the final Apply confirmation can begin a system change.
4. If Gatekeeper requires **Open Anyway**, stop. Do not bypass Gatekeeper; report the release through the documented support or security route.

GitHub Releases remains the canonical download. Homebrew is not offered with the initial publication; a project-owned tap must pass separate install, upgrade, uninstall, and `zap` verification before it is advertised.

Use the [public support form](https://github.com/GGULBAE/desk-setup-switcher/issues/new?template=support_question.yml) for non-sensitive questions. Use the [private vulnerability report](https://github.com/GGULBAE/desk-setup-switcher/security/advisories/new) for vulnerabilities, unsafe mutations, privacy leaks, exposed secrets, or rollback failures; do not disclose those details publicly.

# Desk Setup Switcher v0.1.0 public beta — 한국어

Desk Setup Switcher `v0.1.0`은 책상 프로필을 저장하고 사용자가 명시적으로 적용하는 로컬 전용 macOS 메뉴 막대 앱의 첫 public-beta 버전입니다.

## Capture → Edit → Review & Apply

- Mac 설정을 바꾸지 않고 읽을 수 있는 디스플레이·오디오·네트워크 값을 **Capture**합니다.
- 프로필에 포함할 값만 **Edit**합니다.
- 명시적으로 Apply하기 전에 예정된 작업과 제외 항목을 모두 **Review & Apply**합니다.
- 계정, 클라우드 동기화, 텔레메트리, 분석, 자동 전환, 앱 내 updater 없이 프로필과 민감 정보를 제거한 진단을 Mac 안에 보관합니다.

## 지원 및 검증 범위

- `v0.1.0`의 지원 범위는 **macOS 14 Sonoma deployment target의 Apple Silicon**입니다. 이 지원 주장은 exact signed candidate가 Sonoma 수명주기 검증을 통과한 뒤에만 해당 candidate에 적용되며, 공개 전 draft는 지원을 입증하지 않습니다. 이번 beta는 Intel을 지원하지 않습니다.
- Display, Audio, Network의 apply/rollback 경로는 실제 하드웨어 변경 검증이 아니라 결정론적 mock 검증 상태입니다. 이번 beta는 실제 설정 변경 검증을 주장하지 않습니다.
- Rollback은 최선을 다해 복구를 시도하지만 실패할 수 있습니다. 항목별 결과를 확인하고 현재 macOS 상태를 직접 점검하세요.
- 포괄적인 보조 기술 인증을 주장하지 않습니다. 키보드 동작, 접근성 이름·값, 비색상 상태 단서는 지원 UI 범위에 포함합니다.

## 안전하게 설치하기

Draft, prerelease candidate, Actions artifact 또는 ad-hoc DMG는 지원되는 다운로드가 아닙니다. 전체 [배포 게이트](https://github.com/GGULBAE/desk-setup-switcher/blob/v0.1.0/docs/DISTRIBUTION.md)가 통과된 뒤 maintainer가 승인한 GitHub Release에 첨부된 immutable Developer ID 서명·공증·staple 완료 DMG만 지원되는 artifact입니다.

1. 동일한 승인 GitHub Release에서 DMG와 checksum을 함께 내려받습니다.
2. DMG를 열기 전에 공개된 SHA-256을 확인합니다.
3. DMG에서 앱을 설치하고 메뉴 막대 앱을 실행한 뒤 Capture → Edit → Review & Apply 순서로 사용합니다. Capture는 적용하지 않고 읽기만 하며, 마지막 Apply 확인에서만 시스템 변경을 시작할 수 있습니다.
4. Gatekeeper가 **Open Anyway**를 요구하면 중단하세요. Gatekeeper를 우회하지 말고 문서화된 support 또는 security 경로로 릴리스를 신고하세요.

GitHub Releases가 공식 다운로드 경로입니다. 첫 공개 시점에는 Homebrew를 제공하지 않으며, 프로젝트 own tap의 install, upgrade, uninstall, `zap` 검증이 끝나기 전에는 이를 홍보하지 않습니다.

민감하지 않은 질문은 [공개 지원 양식](https://github.com/GGULBAE/desk-setup-switcher/issues/new?template=support_question.yml)을 이용하세요. 취약점·안전하지 않은 변경·개인정보 유출·노출된 secret·rollback 실패는 [비공개 취약점 신고](https://github.com/GGULBAE/desk-setup-switcher/security/advisories/new)로 제보하고 공개적으로 세부 내용을 남기지 마세요.
